List of recommended software
Software Description Link
Zed Zed is a code editor built in Rust that prioritizes speed and responsiveness. Super clean and minimalist UI that's similar to VS Code, but it's more lightweight and faster. The dev team pushes updates based on user feedback regularly. And it's open source! LINK →
Raycast Raycast is a productivity tool that replaces your Mac's Spotlight with a customizable command center, allowing quick access to apps, files, and services through extensible plugins and keyboard shortcuts. LINK →
GNU Emacs GNU Emacs, possibly the perfect text editor. LINK →
1Password 1Password is the essential tool I recommend to everyone – it securely manages all your passwords and syncs across devices. LINK →
AeroSpace "AeroSpace is an i3-like tiling window manager for macOS". As a tiling window manager proselitizer, it is an absolute necessity for macOS. It's the closest I've gotten to the DWM heaven. LINK →
Notes:
There is no particular order.
If a link is dead or the site is down, please let me know.
I removed `warp.dev` from this list. I enjoyed using it until too many AI features were pushed into the product. I kept accidently triggering a chat, and then having to turn those features off. Half of the features I liked about it can be replicated using PS_PROMPT and PS1 in your shell configuration.
I removed Obsidian from this list. I've been disillusioned by the markdown, Obsidian obsession of late. It is ironically starting to feel less extensible and more bloated. On top of that, the more I learn about org mode, the more it rounds out the rough edges of using markdown for personal knowledge management and overall digital organization. I'll be writing more on this soon.
